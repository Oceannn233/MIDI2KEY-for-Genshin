原琴律桥 · 本地版
=================

启动方法
--------
1. 关闭之前连接过电钢琴的 Web MIDI 网页、旧版 midi2key.py 和 DAW。
2. 双击 run-local.bat。
3. 首次运行会创建独立 Python 3 环境并安装依赖。
4. 浏览器会自动打开 http://127.0.0.1:17321。
5. 在网页里选择 Roland、设置原调和映射策略，再开启“游戏输出”。
6. 切回原神窗口弹奏；需要立刻停止时，在网页点“紧急释放”或按 Ctrl+C。

为什么不会再和网页抢 MIDI 端口
-------------------------------
浏览器页面不再调用 Web MIDI。只有本地 Python 服务打开 Roland 输入端口，
网页通过 127.0.0.1 的本地连接读状态、改参数，因此一个设备只有一个持有者。

端口仍显示被占用时
------------------
关闭其他 MIDI 软件和曾连接 Web MIDI 的浏览器标签页，等待 2 秒后在网页点击
“重新连接”。如仍失败，关闭并重新打开电钢琴电源，再刷新设备列表。
